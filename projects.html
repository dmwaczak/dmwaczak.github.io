YellowDog Coffee Rewards App

A full-stack Flask web application built for a real-world coffee shop to manage customer punch cards and loyalty points. Features include barista dashboard login, QR code scanning, Firebase integration for backend storage, and email confirmations. Designed to streamline in-store operations while enhancing the customer experience.

Note: The live demo is hosted on Render, which may take up to 30 seconds to load if the app has been idle.
